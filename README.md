## Installation

```bash
$ pnpm install
```

## Running the backend app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Running the frontend app

```bash
$ cd frontend
$ npm run start
```

## Queries

A set of sample GraphQL queries can be found in `graphql-queries.json` (Exported from Insomnia Curl).
